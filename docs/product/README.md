# Product & Marketing

Long-form product planning and marketing material. Not implementation plans (those live in [`../superpowers/plans/`](../superpowers/plans)) and not user-facing site content (that lives in [`../../apps/docs/content/docs/`](../../apps/docs/content/docs)).

## Contents

| File | Topic |
|---|---|
| [`product-video-plan.md`](./product-video-plan.md) | Plan for the product video (script, shots, scope) |
| [`product-video-implementation-notes.md`](./product-video-implementation-notes.md) | Implementation notes captured while making the product video |
| [`marketing/blog-content-calendar.md`](./marketing/blog-content-calendar.md) | Editorial calendar for the blog |
| [`marketing/product-hunt-launch.md`](./marketing/product-hunt-launch.md) | Product Hunt launch playbook |

## When to add here

- Long-form plans for product or marketing initiatives that span multiple sessions or weeks.
- Asset planning docs (videos, launches, campaigns).

Things that don't go here:

- Implementation specs → [`../superpowers/specs/`](../superpowers/specs).
- Feature plans → [`../superpowers/plans/`](../superpowers/plans).
- Day-to-day notes → [`../notes/`](../notes).
- Published marketing copy → the marketing site itself.

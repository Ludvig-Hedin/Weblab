# Weblab Explainer — Voiceover Script

Total spoken: ~45 seconds across a 75-second timeline. ~140 words per minute. Calm, confident, mid-low female voice. Never hyped.

ElevenLabs is **not configured** in this build — no API key present at build time. Captions render the script regardless. To add VO later: generate with the ElevenLabs `text-to-speech` skill (or `npx hyperframes tts` for local Kokoro), save to `assets/audio/voiceover-full.mp3`, and the existing `<audio>` clip in `index.html` will play it.

## Pronunciation

- Weblab → *Web-lab*, single stress on first syllable.
- Next.js → *next-jay-ess*.
- AST → letters, *A-S-T*.
- Tailwind → single word, *tail-wind*.
- weblab.build → *Web-lab dot build*.
